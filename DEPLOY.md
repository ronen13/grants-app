## שלב 2 — העלאה ל-GitHub 🐙

GitHub הוא מקום אחסון קוד — Render יקרא ממנו.

1. כנס ל-**https://github.com** → צור חשבון חינמי אם אין לך
2. לחץ **"New repository"** (כפתור ירוק בצד ימין)
3. שם: `grants-in-click` ← הכנס
4. ודא שנבחר **Private** (כדי שהסיסמה לא תהיה גלויה)
5. לחץ **"Create repository"**

**העלאת הקבצים:**
6. בדף שנפתח, לחץ **"uploading an existing file"**
7. גרור את כל הקבצים מהתיקייה (כולל תת-תיקיית `public`)
8. לחץ **"Commit changes"**

## שלב 3 — פתיחת חשבון Render ☁️

1. כנס ל-**https://render.com** → לחץ **"Get Started for Free"**
2. בחר **"Continue with GitHub"** — יתחבר לחשבון שיצרת

---

## שלב 4 — יצירת ה-Web Service ⚙️

1. ב-Render לחץ **"New +"** → **"Web Service"**
2. בחר **"Connect a repository"** → בחר את `grants-in-click`
3. מלא את הפרטים:

| שדה | מה להכניס |
|-----|-----------|
| **Name** | `grants-in-click` |
| **Region** | `Frankfurt (EU)` — הכי קרוב לישראל |
| **Branch** | `main` |
| **Runtime** | `Node` |
| **Build Command** | `npm install` |
| **Start Command** | `npm start` |
| **Instance Type** | `Free` |

4. גלול למטה → לחץ **"Add Environment Variable"** → הוסף:

| Key | Value |
|-----|-------|
| `ADMIN_PASSWORD` | סיסמה חזקה שתבחר (לדוגמה: `Ronen2024!`) |

> ⚠️ **שמור את הסיסמה** — תשתמש בה לכניסה למערכת

5. לחץ **"Create Web Service"**

## שלב 6 — חיבור הדומיין שלך 🌐

ב-Render תקבל כתובת זמנית כמו:
`https://grants-in-click.onrender.com`

**לחיבור הדומיין שלך:**
1. ב-Render → לחץ על השירות → **"Settings"** → **"Custom Domains"**
2. לחץ **"Add Custom Domain"**
3. הכנס את הדומיין שלך (לדוגמה: `app.grants-click.co.il`)
4. תקבל שני רשומות DNS להוסיף:
   - **Type:** CNAME
   - **Name:** `app` (או כל תת-דומיין שבחרת)
   - **Value:** הכתובת שRender נותן

5. כנס לפאנל הדומיין שלך (לרוב: `namecheap.com` / `godaddy.com` / `RegistrarIL`) → DNS Management
6. הוסף את הרשומה → שמור
7. המתן **עד שעה** עד שה-DNS מתפשט
